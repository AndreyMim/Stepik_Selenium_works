# Stepik\_Selenium\_works



*Задание*: запуск автотестов для разных языков интерфейса из модуля 3.6

курса "Автоматизация тестирования с помощью Selenium и Python".



При запуске Pytest можно выбрать браузер (chrome или firefox) и язык интерфейса.

По умолчанию выбирается "--browser\_name=chrome --language=en"



*Пример:* pytest -s -v --browser\_name=firefox  --language=fr test\_items.py
